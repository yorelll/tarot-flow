# 塔罗牌占卜 - Tarot Flow 🔮

一个炫酷的前端塔罗牌抽卡项目，支持手势控制，可直接在浏览器中运行。

## ✨ 功能特点

- 🎴 **精美的塔罗牌设计** - 22张大阿尔卡纳塔罗牌，神秘紫金配色
- 👋 **手势控制** - 使用摄像头识别手势进行卡牌选择
  - 左右移动手掌：选择不同的卡牌
  - 握拳：确认选择当前卡牌
  - 五指张开：查看已选卡牌的牌义
- 🔄 **翻牌动画** - 流畅的3D翻牌效果
- 📖 **牌义解读** - 每张卡牌都有详细的正位和逆位解读
- ⌨️ **键盘控制** - 也可以使用方向键和回车键操作

## 🚀 快速开始

### 直接运行

1. 下载或克隆本项目
2. 双击打开 `index.html` 文件，或使用本地服务器运行

### 使用本地服务器（推荐）

```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js
npx http-server

# 使用 PHP
php -S localhost:8080
```

然后在浏览器中访问 `http://localhost:8080`

## 🎮 操作指南

### 手势控制模式

1. 点击 **"开启摄像头"** 按钮
2. 允许浏览器访问摄像头
3. 将手放入摄像头画面中
4. 使用以下手势进行操作：
   - **左右移动手掌** - 选择卡牌
   - **握拳并保持** - 确认选择当前高亮的卡牌
   - **五指张开并保持** - 查看最近选择卡牌的牌义

### 键盘控制模式

- **← / →** - 左右选择卡牌
- **Enter / Space** - 确认选择

### 鼠标控制

- 点击卡牌组中的卡牌进行选择
- 点击已选区域的卡牌查看牌义
